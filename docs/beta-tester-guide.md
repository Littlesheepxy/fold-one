# 知更封闭内测指南

给内测同学与发放内测码的人用。官网申请入口：`/beta`（申请内测码）。

## 当前能测什么 / 别指望什么

| 主推（请认真测） | 先别当卖点 |
|------------------|------------|
| 右 ⌘ 短按：口述 → 整理后插入 | 云端热词「开箱必准」 |
| 右 ⌘ 长按：情境代回（多草案 → 插入） | 日程智能（默认关） |
| ⌥ Space：交给本机 Agent（需自备 CLI） | 跨应用 Ultra 自动编排 |
| 本地记忆 / 工作轨迹 | 自动更新与崩溃上报（尚未接线） |
| 外发/授权前的确认卡 | 签名 DMG 商店分发（见下） |

**默认安全**：Agent Subagent、本地脚本、UI-TARS 默认关。内测请保持关闭，除非你明确在测 Agent。

## 安装（现状）

正式签名 DMG / 公证流程尚未合入仓库。当前封闭内测两种路径：

### A. 开发者自建（技术同学）

```bash
pnpm install
cp .env.example .env   # 填入 ASR / Planner Key，或走本地 Whisper + 有限智能体验
pnpm asr:dev           # 若用云端 ASR
pnpm desktop:dev
```

首次启动按引导开启：**辅助功能**、**麦克风**；热键以引导为准（右 ⌘ / ⌥ Space）。

### B. 预装包（非技术同学）

由我们私下发送安装包与内测码说明。若 Gatekeeper 提示未验证开发者：系统设置 → 隐私与安全性 → 仍要打开（仅限我们发出的包）。

## 快捷键（以产品为准）

| 操作 | 快捷键 |
|------|--------|
| 结构化输入 | 右 ⌘ **短按** |
| 情境代回 | 右 ⌘ **长按** |
| Agent | **⌥ Space** |
| 取消 | Esc |

> README 旧文若仍写「⌥ Space 开关语音」已过时，以本表与引导页为准。

## 权限清单

1. 麦克风 — 语音输入  
2. 辅助功能 — 读窗口/插入文字  
3. 屏幕录制 — 代回/OCR 等需要时再开（连接页可跳转）  
4. 日历 — 默认关闭，需要时再开  

## 反馈怎么交

1. 应用内：设置 → 高级 → **反馈问题**（打开邮件）  
2. 或发至 [hello@zhigeng.app](mailto:hello@zhigeng.app)  
3. 主题建议：`知更内测反馈 - 一句话现象`  
4. 附上：macOS 版本、复现步骤；可用设置里 **打开数据目录** 自行查看本地库（勿把整个目录+密钥打包外传）

测完语音/HITL 后，开发侧可跑：

```bash
pnpm exec tsx scripts/read-stress-log.ts --since=30m
```

## 发放内测码前检查清单（运营）

- [ ] 对方已同意用户协议与隐私政策（申请表勾选）  
- [ ] 说明「主推代回/转写，Agent 实验」  
- [ ] 给反馈邮箱/群  
- [ ] Subagent 默认关（勿引导一上来开）  
- [ ] 签名包就绪前，只发给能跑 `pnpm desktop:dev` 的人，或你代为打包
