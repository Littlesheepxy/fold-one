## 怎么跑

```bash
# 可选：指定一条意图自动进 executeTask
FOLD_E2E_INTENT="帮我整理刚下载的报价发给 Jason" pnpm desktop:dev

# 或启动后 DevTools Console：
# await window.fold.runTask("你的意图")
```
