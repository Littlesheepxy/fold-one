# 知更品牌资产生成规范

给设计师 / AI 出图时按此交付，开发侧用 `pnpm --filter @fold/desktop build:icons` 自动生成各尺寸。

---

## 需要几套图？

| 用途 | 文件名建议 | 色彩 | 背景 |
|------|-----------|------|------|
| **主源图（彩色）** | `zhigeng-robin-master.png` | 彩色知更鸟 | 透明或纯色均可 |
| **主源图（白色剪影）** | `zhigeng-robin-mono.png` | 纯白剪影知更鸟 | **透明** |
| **Dock 应用图标** | 由脚本合成 | 彩色鸟 + 蓝紫渐变 squircle | 脚本自动加圆角底板 |
| **菜单栏 tray** | `zhigeng-tray-template.png`（可选精修） | **纯白剪影** | **透明** |
| **悬浮球 / 语音条** | `zhigeng-mark-mono.png`（可选精修） | **纯白剪影** | **透明** |
| **侧栏品牌** | `zhigeng-mark.png` | 彩色鸟 | 透明 |

**结论：悬浮球和菜单栏建议用黑白（白剪影），Dock 和侧栏用彩色。**

当前无精修稿时，脚本会用 `filter: invert` 从主图临时生成 mono 版；你提供更干净的剪影后替换即可。

---

## 1. 主源图 `zhigeng-robin-master.png`

- **尺寸**：1024×1024 px（或更大，1:1）
- **格式**：PNG，真 PNG（不要 JPEG 改后缀）
- **内容**：知更鸟侧脸，占画布约 **70–80%**，四周留均匀边距
- **背景**：**透明** 最佳；若带浅色底，Dock 合成时会裁进 squircle
- **风格**：3D 黏土 / 柔和渐变，蓝紫主体 + 白色胸腹 + 橙色喙

---

## 2. Dock 图标（脚本自动合成，无需单独出图）

- 画布 1024×1024
- 圆角矩形（squircle）圆角半径约 **196px**（相对 1024）
- 底板：**纯白 `#FFFFFF`** squircle（与系统浅色图标一致）
- 鸟缩小至内区约 **688px**，居中
- 不要外圈白边；不要正方形白底

---

## 3. 菜单栏 Tray `zhigeng-tray-template.png`

- **尺寸**：36×36 px（@2x 源图），开发会再导出 18px
- **颜色**：**纯 `#FFFFFF` 剪影**（macOS 菜单栏 template 图标）
- **背景**：**完全透明**
- **造型**：只保留鸟头 + 喙的 **简化轮廓**，线条粗、细节少（18pt 下可辨）
- **禁止**：渐变、阴影、彩色、方形底

---

## 4. 悬浮球 Mono `zhigeng-mark-mono.png`

- **尺寸**：128×128 px（UI 显示约 22–28px）
- **颜色**：**纯白 `#FFFFFF`** 剪影
- **背景**：**透明**
- **造型**：与 tray 同系列简化鸟形，可略饱满（比 tray 大一点）
- **场景**：深色玻璃胶囊 `rgb(8 9 12 / 0.62)` 上显示，需高对比

若希望悬浮球用**彩色鸟**，另交付 `zhigeng-mark-color.png`（透明底，128px），并在设置里切换；默认推荐 mono。

---

## 5. 侧栏彩色 Mark `zhigeng-mark.png`

- **尺寸**：128×128 px
- **彩色**知更鸟，透明底
- 用于设置页侧栏「知更」品牌区

---

## 检查清单

- [ ] 主图是透明底 PNG
- [ ] Tray / 悬浮 mono 是 **白色剪影**，不是灰度图
- [ ] 小尺寸（18px）仍能认出是鸟
- [ ] Dock 预览无白方块、圆角与系统图标一致

交付后放进 `apps/desktop/public/brand/`，运行：

```bash
pnpm --filter @fold/desktop build:icons
```
